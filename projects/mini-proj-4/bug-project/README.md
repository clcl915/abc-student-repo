# hehe doraemon
A browser extension that populates any web page with a hehe Doraemon sticker.


Install this on your friends browsers to prank them. The doraemons won't go away until they find and click on the clown icon.  

Drew inspiration from [Cornify](https://chrome.google.com/webstore/detail/cornify-unicorn-and-rainb/ghdnfbmfflgelndafnlgabneckbmfpla?hl=en) !
